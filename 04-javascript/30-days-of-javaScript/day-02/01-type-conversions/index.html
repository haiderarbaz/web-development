Type Conversions:

  • The three most widely used type conversions are to string, to number, and to boolean.

  • 1. String Conversion

    • String conversion happens when we want to turn something into text.

    • You can do this with String(value) - just put whatever you want to convert inside the parentheses.

    • Example:
      1. String(123) becomes "123"
      2. String(true) becomes "true"
      3. String(false) becomes "false"
      4. String(null) becomes "null"

  • 2. Numeric Conversion

    • Numeric conversion in mathematical functions and expressions happens automatically. Except while doing Addition (We have to do it explicitly)

    • To explicitly convert a value to a number we use (Number(value))

    • Explicit conversion is usually required when we read a value from a string-based source like a text form but expect a number to be entered.

    • NOTE:

      • null and undefined behave differently in numeric conversions

      • null becomes zero while undefined becomes NaN.

  • 3. Boolean Conversion

    • Boolean conversion is the simplest one.

    • It happens in logical operations

    • But can also be performed explicitly with a call to Boolean(value).

    • Boolean Conversion Rules:

      1. Values that are intuitively “empty”, like 0, an empty string, null, undefined, and NaN, become false.

      2. Other values become true.
