There are 8 basic data types in JavaScript.

  • Seven primitive data types:

    1. Number

      • Number type represents both integer and floating point numbers.

      • Integers are limited by ±(253-1)

      • NaN represents a computational error. It is a result of an incorrect or an undefined mathematical operation

      • NaN is sticky. Any further mathematical operation on NaN returns NaN

      • Only one exception to that: (NaN ** 0 is 1)

    2. BigInt

      • A bigInt value is created by appending n to the end of an integer

      • bigInt for integer numbers of arbitrary length.

      • const bigInt = 1234567890123456789012345678901234567890n;

    3. String

      • String for strings. A string may have zero or more characters, there’s no separate single-character type.

      • String in JavaScript must be surrounded by quotes.

      • There are three types of quotes:

        1. Double quotes: "Hello"

        2. Single quotes: 'Hello'

        3. Backticks: `Hello`

    4. Boolean

      • The boolean type has only two values:

        1. true
        2. false

      • This type is commonly used to store yes/no values: true means “yes, correct”, and false means “no, incorrect”.

    5. Null

      • In JavaScript, Null is not a refrence to a non-existing object or a null pointer like in some other language.

      • Here, in JavaScript, Null is just a special value which represents nothing, empty or unknown value.

      • And, it can be anytype.

    6. Undefined

      • Undefined meaning is "Value is not assigned."

      • If a variable is declared but not assigned then it's value is undefined.

    7. Symbol

      • Symnbol type is used to create unique "identifiers" for objects

  • And one non-primitive data type:

    1. Object

      • Objects are used to store collection of data in key value pair and more complex properties.

  • The typeOf operator

    • Typeof operator return a string indicating the type of operand value.

    • It's useful when you want to process values of different types differently or just want do a quick check.
