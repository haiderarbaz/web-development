Variables and There Types

What is Variables?

  • Variable is named memory location to store data.

  • We use variable to store goodies, visitors and other data.

Variable Naming

  • There are two limitations on variable names in JavaScript:

    1. The name must contain only letters, digits, or the symbols $ and _.

    2. The first character must not be a digit.

Types of Variables

  1. var

    • Var is a function or global scope.

    • It means var can be accessed through out the function if they are declared in or globally, if declared outside the function

    • Var can be re-assigned and re-dclared.

  2. let

    • Let is a block or local scope.

    • It means let cannot be accessed outside the scope.

    • Let can be re-assigned but cannot be re-dclared.

  3. const

    • Const is a also a block or local scope.

    • It also cannot be accessed outside the scope.

    • Const cannot be re-assigned after the intial declaration.

    • Const should must be intialized at the time of declaration otherwise it will give a error message ("'const' declarations must be initialized.")
